## ✨ 核心功能亮点

- 🎨 **色彩可视化管理**：直观的颜色展示，支持色号、色系分类，告别盲目翻找。
- 📦 **精准库存追踪**：实时记录每种颜色的剩余数量，用了多少、还剩多少一目了然。
- ⚠️ **智能缺货预警**：库存低于阈值自动提醒，再也不用担心拼到一半没豆子。
- 📊 **出入库明细记录**：清晰的流水账单，方便复盘和成本核算。
- 💾 **纯本地极速存储**：数据安全存储在浏览器本地（localStorage），**无需服务器和数据库**，极速响应。
- 📱 **响应式极简设计**：电脑、手机、平板完美适配，随时随地管理你的“色彩金库”。

## 🛠️ 硬核技术栈

本项目采用目前最流行、最前沿的纯前端架构，代码结构清晰，极具学习和二次开发价值：

### 前端架构 (Pure Frontend)
- **核心框架**：React 18 + TypeScript (Vite 构建)
- **UI 组件库**：shadcn/ui + Tailwind CSS (极简现代，颜值极高)
- **状态管理 & 路由**：React Router DOM + React Query
- **图标库**：Lucide React
- **数据持久化**：LocalStorage (无需配置后端，部署成本为 0)
